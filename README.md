# Azure-Stream-Analytics
This project demonstrates a real-time data processing pipeline using Microsoft Azure services. It collects data from IoT devices, processes it using Azure Stream Analytics, and visualizes insights through Power BI dashboards.

Key Features

Real-time data ingestion using Azure IoT Hub

Stream processing with Azure Stream Analytics

Data visualization using Power BI

End-to-end pipeline from device to dashboard

Tech Stack

Microsoft Azure (IoT Hub, Stream Analytics)

Power BI

Cloud-based data processing

Architecture

IoT devices send real-time data to Azure IoT Hub

Stream Analytics processes incoming data streams

Processed data is sent to Power BI

Dashboards display real-time insights

Results

Successfully processed and visualized real-time data streams

Enabled monitoring and analysis through interactive dashboards
